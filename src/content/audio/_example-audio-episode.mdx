## About this template

This MDX file demonstrates **every field** available in the `audio` collection schema.  
When creating a real episode:

- Change `provider` to match your source and fill the corresponding URL/ID field:
  - `self` → set `contentUrl`
  - `spotify` → set `spotifyUrl`
  - `apple` → set `applePodcastsUrl`
  - `soundcloud` → set `soundcloudUrl`
  - `youtube` → set `youtubeId`
  - `embed` → set `embedUrl`
- Update `slug`, `publishDate`, `durationSeconds`, `thumbnails`, and `chapters`.
- Set `draft: false` and `index: true` when publishing.
